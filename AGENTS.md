# Project Guidelines

## Student-Friendly Code
This project is built for students — prioritize simplicity and straightforwardness over abstraction or clever patterns.

## AI Model
The AI model used in this project is **Google Gemini**.

## Friendly Error Messages
Errors should be caught gracefully and rendered on the page in a friendly, human-readable format so users understand what went wrong.

## Surgical Precision
The fix you are going to implement are to be done with surgical precision. If you have any major decision(s) to make, do not fail to ask questions.

## Code Fix Reports
Whenever fixing code, provide a detailed report with:
1. **What went wrong** — exactly what the issue was and the root cause.
2. **Fixes implemented** — what changes were made and why.
3. **Risks found** — any potential side effects, edge cases, or regressions introduced.
